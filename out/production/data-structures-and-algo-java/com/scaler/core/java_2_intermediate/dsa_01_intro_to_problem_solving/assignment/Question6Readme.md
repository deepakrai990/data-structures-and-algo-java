### Q6. Sum of N natural numbers

#### Problem Description
```text
The sum of n natural numbers is :
```
#### Options
```text
1. (n*(n+1))/2
2. (n*(n-1))/2
3. (n+1)/2
4. (n*(n+1)*(2*n+1))/6
```

* Complete Solution
```text
As the first n natural numbers are 1,2,3,4,5,6,.........n
They form an AP with a=1 and d=1
The sum of AP for n numbers = ((2*a + (n-1)*d)*n)/2
=((2 + n - 1)*n)/2
=(n*(n+1))/2
```
