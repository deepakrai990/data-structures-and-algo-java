## Q1. Total Number of Subarrays

#### Problem Description
```text
Given an array of size 21, find the total number of 
subarrays of the array.
```

### Options
```text
1. 254
2. 231
3. 369
4. 441
```

* Complete Solution
```text
Total number of subarrays of an array of size N is
N*(N + 1)/2
So, 21 * (21 + 1) / 2 = 21 * 11 = 231
```
