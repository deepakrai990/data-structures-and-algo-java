### Q6. Number of elements in a range - II
#### Problem Description
```text
Total number of elements between (a b) (excluding a and b) is 
given by which of the following formulas?
```
#### Options
```text
1. b - a

2. b - a - 1

3. b - a + 1

4. b - a - 2
```

* Complete Solution
```text
Total number of elements between (a,b) is given
by b-a-1
```
