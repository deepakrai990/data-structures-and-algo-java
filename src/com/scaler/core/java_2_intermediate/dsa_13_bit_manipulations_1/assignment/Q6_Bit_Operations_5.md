### Q6. Bit Operations - 5
#### Problem Description
```text
What are the values of a&a, a|a, a^a?
```
#### Options
```text
1. a&a = a , a|a = a , a^a = a
2. a&a = a , a|a = a , a^a = 0
3. a&a = 0 , a|a = a , a^a = a
4. a&a = 0 , a|a = a , a^a = 0
```

* Complete Solution
```text
a&a = a
a|a = a
a^a = 0
```
