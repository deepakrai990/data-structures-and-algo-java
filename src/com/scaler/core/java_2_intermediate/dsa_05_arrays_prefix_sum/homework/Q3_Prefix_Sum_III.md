### Q3. Prefix Sum - III
#### Problem Description
```text
Given the prefix sum of an array, prefSum = [-2, 4, 1, 5, 2]

What is the sum of the original array from index 1 to 2 (0-based) ?
```
#### Options
```text
1. 5
2. -3
3. 3
4. -5
```

* Complete Solution
```text
The original array is [-2, 6, -3, 4, -3].
The sum in the range [1, 2] is 3.
```
