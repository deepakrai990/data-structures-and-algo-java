### Q1. Prefix Sum - I
#### Problem Description
```text
Given the prefix sum of an array, prefSum = [-2, 4, 1, 5, 2]

What is the sum of the original array from index 0 to 2 (0-based) ?
```
#### Options
```text
1. 3
2. 4
3. 1
4. 5
```

* Complete Solution
```text
The original array is [-2, 6, -3, 4, -3].
The sum in the range [0, 2] is 1.
```
