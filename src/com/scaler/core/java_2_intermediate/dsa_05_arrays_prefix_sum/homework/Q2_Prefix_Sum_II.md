### Q2. Prefix Sum - II
#### Problem Description
```text
Given the prefix sum of an array, prefSum = [-2, 4, 1, 5, 2]

What is the sum of the original array from index 2 to 4 (0-based) ?
```
#### Options
```text
1. 2
2. 1
3. -2
4. 8
```

* Complete Solution
```text
The original array is [-2, 6, -3, 4, -3].
The sum in the range [2, 4] is -2.
```
