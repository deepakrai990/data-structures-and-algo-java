### Q2. Time Complexity of Prefix Sum
#### Problem Description
```text
What is the time complexity of creating the prefix sum array of 
an array A of N integers ?
```
#### Options
```text
1. O(1)
2. O(N)
3. O(N^2)
4. O(N^3)
```

* Complete Solution
```text
The time complexity of creating the prefix sum array is O(N).
```
