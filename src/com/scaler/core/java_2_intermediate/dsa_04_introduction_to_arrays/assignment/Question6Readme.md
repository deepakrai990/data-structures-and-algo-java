### Q6. Time Complexity - Arrays
#### Problem Description
```text
What is the time complexity for inserting/deleting at the beginning of the array?
```
#### Options
```text
1. O(1)
2. O(N)
3. O(logN)
4. O(N^2)
```
* Complete Solution
```text
Insertion and deletion at the beginning of the array is O(N)
```
