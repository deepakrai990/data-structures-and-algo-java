### Q3. Find Time Complexity - 3
#### Problem Description
```text
What is the time complexity of the following code snippet

int func(int n){
    int s = 0;
    for(int i = 1 ; i <= 100 ; i++){
        s = s + i;
    }
    return s;
}
```
#### Options
```text
1. O(n)
2. O(n^2)
3. O(s)
4. O(1)
```

* Complete Solution
```text
The answer is O(1). 
The loop runs for 100 times.
```
