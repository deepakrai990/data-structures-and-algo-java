### Q7. Find Iterations
#### Problem Description
```text
Find the number of times below code runs where N is a perfect square

for i -> 1 to N

    if(i * i == N)

        return i
```
#### Options
```text
1. N
2. N/2
3. sqrt(N)
4. log N
```

* Complete Solution
```text
The code runs for sqrt(N) times.
```
