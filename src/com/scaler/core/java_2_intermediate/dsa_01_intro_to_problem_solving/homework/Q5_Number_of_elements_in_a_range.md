### Q5. Number of elements in a range
#### Problem Description
```text
How many elements are there between [135, 246] (inclusive of them) ?
```
#### Options
```text
1. 111

2. 112

3. 113

4. 114
```

* Complete Solution
```text
The number of elements in the range[a,b] is
b - a + 1
246 - 135 + 1 = 112
```
