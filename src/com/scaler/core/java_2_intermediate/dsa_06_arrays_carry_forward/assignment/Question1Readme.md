### Q1. Find all the sub-arrays
#### Problem Description
```text
Given A = [1, -2, 5, 2, -1, 6]. Choose all the valid subarrays of the array
```
#### Options
```text
1. [5, 2, -1]
2. [-2, -1, 6]
3. [1, -2, 5, 2]
4. [6]
```

* Complete Solution
```text
The valid sub-arrays are [5, 2, -1], [1, -2, 5, 2]
and [6].
```
