### Q3. Special Subsequences "AG"
#### Problem Description
```text
You have given a string A having Uppercase English letters.

You have to find how many times subsequence "AG" is there in the given string.

NOTE: Return the answer modulo 10^9 + 7 as the answer can be very large.
```
<p>
    <strong>NOTE: </strong> 
    Return the answer modulo <strong>10<sup>9</sup> + 7</strong> 
    as the answer can be very large.
</p>

#### Problem Constraints
<div style="background-color: #f9f9f9; padding: 5px 10px; ">
    <p>1 &lt;= length(A) &lt;= 10<sup>5</sup></p>
</div>

```text
1 <= length(A) <= 10^5
```
#### Input Format
```text
First and only argument is a string A.
```
#### Output Format
```text
Return an integer denoting the answer.
```
#### Example Input
```text
Input 1:
 A = "ABCGAG"

Input 2:
 A = "GAB"
```
#### Example Output
```text
Output 1:
 3

Output 2:
 0
```
#### Example Explanation
```text
Explanation 1:
 Subsequence "AG" is 3 times in given string 

Explanation 2:
 There is no subsequence "AG" in the given string.
```
